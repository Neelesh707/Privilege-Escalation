# Privilege Escalation

This repository documents practical privilege escalation techniques, enumeration methodologies, exploitation workflows, and mitigation strategies for Linux and Windows operating systems. The content is based on hands-on laboratory environments, Capture The Flag (CTF) challenges, real-world vulnerability research, and authorized penetration testing exercises.

## Scope

Topics in this repository may include, but are not limited to:

### Linux
- Enumeration Methodology
- SUID & SGID Binaries
- Linux Capabilities
- Writable Files & Directories
- Cron Jobs
- PATH Hijacking
- Environment Variable Abuse
- NFS Misconfigurations
- Weak File Permissions
- Credential Discovery
- Kernel Exploitation
- GTFOBins
- Container Escapes

### Windows
- Enumeration Methodology
- Weak Service Permissions
- Unquoted Service Paths
- DLL Hijacking
- Registry Misconfigurations
- Scheduled Tasks
- AlwaysInstallElevated
- Token Impersonation
- UAC Bypass
- Credential Discovery
- LOLBAS
- Windows Kernel Exploitation

## Purpose

The objective of this repository is to document structured privilege escalation methodologies, understand common operating system misconfigurations, demonstrate practical exploitation techniques, and provide mitigation recommendations based on security best practices.

All content is intended for educational purposes only. The techniques documented here were performed in controlled laboratory environments, Capture The Flag (CTF) platforms, or systems where explicit authorization was granted. Do not attempt these techniques on systems without permission.
